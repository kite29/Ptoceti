Ptoceti

An Intranet Of things hub framework

Based on open projects

Ptoceti is build from open source projects an standards: Obix as the model for exchanging data, Osgi R4 for the container, Restlet for the Rest implmentation, and so on ...

Deploy on what you like

Develop on OsX, Linux, Windows and deploy on small ARM target. Ptoceti can run on something as small as a Raspberry' Pi.

Connect what you want

Use existing bundles to connect to devices or build your own.
